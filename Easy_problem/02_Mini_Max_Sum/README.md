# Mini Max Sum

## Problem
Given five integers, calculate the minimum and maximum values by summing exactly four integers.

## Example

Input:
1 2 3 4 5

Output:
10 14

## Approach
1. Calculate the total sum of all elements.
2. Subtract each element one by one from the total.
3. Keep track of the minimum and maximum sum.
4. Print the result.

## Time Complexity
O(n)

## Space Complexity
O(1)
