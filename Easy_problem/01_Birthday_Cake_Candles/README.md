# Birthday Cake Candles

## Problem
Given the heights of candles, determine how many candles are tallest.

## Example

Input:
3 2 1 3

Output:
2

## Approach
1. Find the maximum candle height.
2. Count how many times the maximum height appears.
3. Return the count.

## Time Complexity
O(n)

## Space Complexity
O(1)
